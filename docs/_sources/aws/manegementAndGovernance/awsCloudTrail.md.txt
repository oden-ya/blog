# AWS CloudTrail

## 概要
- ユーザーアクティビティやAPI操作を証跡に記録するサービス
- CloudTrailで取得できる書生期は管理イベントとデータイベントがある
- 90日間は無料で証跡を保管することができ、長期保管する場合はS3を利用する

## 主な機能/特徴
- **管理イベント**：AWSリソースの管理操作に関するイベント（EC2インスタンスの起動、IAMポリシーの更新）
- **データイベント**：特定のリソースに対するデータ操作に関連するイベント（S3バケット内のオブジェクトへのPut,Get,Delete操作）
- **CloudTrail Insights**：APIコール及びエラー率に関する異常なアクティビティを特定することができる
