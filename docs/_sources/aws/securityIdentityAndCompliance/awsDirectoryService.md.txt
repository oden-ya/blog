# AWS Directory Service

## 前提
- **ディレクトリサービス**：ネットワーク上のリソースや所在などを管理し、探索するためのサービス
- **Active Directory**：リソースに対するアクセスを認可してくれるサービス

## 概要
- AWS上でディレクトリサービスを利用することができる

## 主な機能/特徴
- **AD connector**：IAMのディレクトリリクエストをオンプレミスのMicrosoft Active Directoryへとりダイレクトするのに使用するディレクトリゲートウェイ
- **Simple AD**：Samba 4 Active Directory Compatible Serverを使用するスタンドアロンのマネージド型ディレクトリ
- **AWS Managed Microsoft AD**：AWSクラウド内にマネージド型Active Directoryを設置
