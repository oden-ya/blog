Amazon Detective¶

概要¶

  • ログデータを収集し、機械学習、統計的分析を用いて、詳細な調査を行う

  • Amazon GuardDuty、AWS Security Hub、CloudTrailログ、VPC Flow Logsと統合することができる

  • IAMユーザーが普段と違うIPから利用した場合や、異常なEC2インスタンスが複数起動した場合などを調査することができる
