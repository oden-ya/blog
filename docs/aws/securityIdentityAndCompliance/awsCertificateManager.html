AWS Certificate Manager(ACM)¶

概要¶

  • ウェブサイトやアプリケーションを保護するSSL/TLS X.509証明書、およびキーの作成、保存、更新を行う

  • CloudFront、Elastic Load Balancing、API Gatewayに証明書をインポートして登録できる

  • ACMは転送中のデータを暗号化する（Data in transit）

  • クライアントからSSL端点（ELBなど）までの通信を暗号化する。ELBからバックエンドまでの暗号化をする場合は、追加でバックエンドに証明書をインストールする必要がある。（バックエンド暗号化）
